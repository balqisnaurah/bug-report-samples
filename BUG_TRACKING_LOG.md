\## Definisi Status



| Status | Deskripsi |

|--------|-----------|

| Open | Bug baru ditemukan, belum di-assign ke developer |

| In Progress | Sedang dikerjakan oleh developer |

| Resolved | Sudah diperbaiki, menunggu verifikasi QA |

| Closed | Sudah diverifikasi QA dan dinyatakan fixed |

| Reopened | Bug muncul kembali setelah dinyatakan fixed |

| Rejected | Bukan bug, atau tidak akan diperbaiki |
